# Background Removal

A web-based tool to remove or replace the background of uploaded images. Users can easily edit and download images with the following features:

## Features
- Remove background from any uploaded image
- Replace background with custom images or colors
- Adjust foreground and final image brightness
- Scale the foreground for better composition
- Download the final edited image

## How to Use
1. Upload an image.
2. Remove or replace the background.
3. Adjust brightness and scale if needed.
4. Download the final image.
